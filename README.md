# WindConvert — Landing Page

> The public landing page for [WindConvert](https://wind-convert.vercel.app/) — a free, no-login CSS ↔ Tailwind converter tool.

---

## What is WindConvert?

WindConvert is a free web tool that converts code in two directions:

- **HTML + CSS → Tailwind CSS** — paste your markup and stylesheet, get back clean Tailwind classes. Output in plain HTML or JSX, your choice.
- **Tailwind CSS → HTML + CSS** — paste Tailwind HTML, get back a separated HTML file and a clean CSS file.

No accounts. No setup. No AI fees. Paste, convert, copy, done.

The conversion engine is fully rule-based — 1,159 mappings covering flexbox, grid, spacing, colors, shadows, borders, and more. Runs server-side in milliseconds.
